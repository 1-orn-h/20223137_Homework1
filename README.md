## 과제 목표

다음 리눅스 명령어에 대해서 조사해 작성할 예정입니다
|  Name  |      Fuction         |
|  :-:   |         ---          |
|  top |시스템의 실시간 성능 확인 가능|
|  ps  |현재 실행중인 프로세스 상태 확인|
|  jobs  |쉘에서 실행된 백그라운드 작업 목록을 보여주는 기능|
|kill |  프로세스에 시그널(signal)을 보내 프로세스를 제어하는 기능|

## 1. top 명령어
+ 용도 : 실시간으로 시스템의 전반적인 상태(CPU, 메모리)와 프로세스들의 자원 사용량 확인
+ 특징 : 실행하면 화면이 3초마다 계속 갱신되는 동적인 명령어
+ 자주 사용하는 키
    + q : 종료
    + P : CPU 사용률 순으로 정렬(기본값)
    + M : 메모리 사용률 순으로 정렬
